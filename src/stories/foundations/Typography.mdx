# Typography

The type scale is density-optimised for an editor UI.
Font family is **Inter Tight** (variable, roman + italic) with system fallbacks.
All sizes are defined as `@theme inline` custom properties in `src/index.css`.

## Weights

Inter Tight is a variable font supporting weights 100–900.
The editor uses regular (400) for body and medium (500) for emphasis.
