# Typography

Mozaicon's type system uses **Inter Tight**, a variable font optimized for
dense UI. All sizes, line-heights, and font families are stored as CSS
custom properties in `src/index.css` and exposed via Tailwind's `@theme inline`.

## Type Scale

Each specimen below reads its values from CSS custom properties at runtime
via `getComputedStyle`, so what you see matches the tokens that ship in production.

## Token Reference

| Utility     | CSS Variable  | Size             | Line Height |
| ----------- | ------------- | ---------------- | ----------- |
| `text-2xl`  | `--text-2xl`  | 2.125rem (34px)  | 1.1         |
| `text-xl`   | `--text-xl`   | 1.125rem (18px)  | 1.3         |
| `text-lg`   | `--text-lg`   | 0.875rem (14px)  | 1.4         |
| `text-base` | `--text-base` | 0.8125rem (13px) | 1.5         |
| `text-sm`   | `--text-sm`   | 0.6875rem (11px) | 1.3         |
| `text-xs`   | `--text-xs`   | 0.625rem (10px)  | 1.2         |

## Theme Support

Toggle the theme switcher above to see how the specimens render in light
and dark mode. Typography tokens are theme-invariant — only color tokens
change between themes. The specimens use `var(--foreground)` for text color,
which resolves to the correct value for each theme.
