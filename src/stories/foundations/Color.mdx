# Color

The palette is built on OKLCH and stored as CSS custom properties in `src/index.css`.
Light theme lives on `:root`; dark overrides apply under the `.dark` class.
Toggle themes with the toolbar above to see both palettes live.

## Surfaces

Five-step surface scale collapsed onto shadcn role names.
Each step is one shade lighter/darker than the last, creating a subtle depth hierarchy without shadows.

## Brand (Indigo)

A single accent color communicates active tool, selection, focus ring, and slider thumb.
Indigo 500 is the anchor; hover/muted/subtle variants shift lightness for context.
