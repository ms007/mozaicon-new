# Spacing

Spacing uses Tailwind's 4 px base grid (`--spacing`).
The DS restricts the scale to a deliberate subset — `p-5` and `p-7` are intentionally skipped to maintain visual rhythm.

## Intentional gaps

`p-5` (20 px) and `p-7` (28 px) are avoided.
Skipping these steps enforces a tighter rhythm in the dense editor chrome and reduces decision fatigue — there is always exactly one "right" size to reach for.
