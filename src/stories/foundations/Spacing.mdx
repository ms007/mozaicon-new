# Spacing

The spacing scale is built on Tailwind CSS v4's `--spacing` token (`0.25rem` = 4px at
default root font size). Every value is a multiple of this base unit, keeping layouts
consistent and predictable.

Use the Tailwind utility classes — `p-4`, `gap-2`, `m-8`, etc. — rather than
hard-coding pixel values.

## Special values

| Utility | Value | Use case                  |
| ------- | ----- | ------------------------- |
| `px`    | 1px   | Borders, fine adjustments |
| `0`     | 0     | Resetting margins/padding |

## Usage guidelines

- **4 / 8 / 12 / 16** — workhorse values for component internals (padding, gaps).
- **20 – 32** — section-level spacing between groups of content.
- **40+** — page-level breathing room; use sparingly.
- Prefer even steps (2, 4, 8, 12, 16…) for visual consistency; half-steps (0.5, 1.5, 2.5, 3.5) are for fine-tuning.
