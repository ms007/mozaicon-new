# Radii

The radius system uses a single `--radius` knob in `:root` that retints the
whole app. All other radius tokens derive from it via `calc()`.

| Token         | Formula                     |
| ------------- | --------------------------- |
| `--radius-sm` | `calc(var(--radius) - 4px)` |
| `--radius-md` | `calc(var(--radius) - 2px)` |
| `--radius-lg` | `var(--radius)`             |
| `--radius-xl` | `calc(var(--radius) + 4px)` |
