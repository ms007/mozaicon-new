# Mozaicon

Pixel-perfect icon design with a keyboard-first workflow and instant SVG export.

## Design Principles

- **Atomic state** — small, focused Jotai atoms; derived atoms for computed values.
- **Command pattern** — every mutation is undoable via the command system.
- **Tokens over magic values** — colours, radii, and spacing come from design tokens wired through Tailwind.
- **Primitives seam** — shadcn/ui components are vendored and wrapped; app code imports from `primitives/`.
